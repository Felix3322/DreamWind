【Automatic Installation】
Drag the CS2Konc_CFG folder into the CFG folder and double-click InstallCS2Konc_CFG.bat for automatic installation.

════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════

【Manual Installation】
Drag the CS2Konc_CFG folder into the CFG folder.

Loading Methods
Method 1
Add the following to the launch options:
+exec CS2Konc_CFG/CSKoncMod
Method 2
Create an autoexec.cfg file and write:
exec CS2Konc_CFG/CSKoncMod 
(Recommended to use autoexec.cfg)
Example path for autoexec.cfg:
【*\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg】

Text Installation: Drag the files (.txt) from the resource folder into CS2's resource folder.  
Example path:  
【*\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\resource】

════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════

【CFG Settings】
In the config folder, there are:
(KeyPreference.cfg) to customize and modify the initial key binding settings of the entire CFG
(OptPreference.cfg) to customize and modify the initial settings of the entire CFG
(PersonalSettings.cfg) to customize and modify personal settings
Please modify the CFG settings according to the text

════════════════════════════════════════════════════════════════════════════════════════════════════════════════════════

【Uninstall】
Double-click UninstallCS2Konc_CFG.bat to uninstall (CS2Konc_CFG)
Then enter the game and reset the keys twice.
OR
bind mouse_x yaw;bind mouse_y pitch;unbindall;binddefaults;binddefaults;joy_response_move 1;joy_side_sensitivity 1.000000;joy_forward_sensitivity 1.000000;cl_scoreboard_mouse_enable_binding +attack2;cl_quickinventory_filename radial_quickinventory.txt;host_writeconfig

【Usage Instructions】
Hold down Right Shift to open the wheel, move the mouse to select a function, and release Right Shift to confirm the selection.  
The scroll wheel can be used to scroll up and down.
